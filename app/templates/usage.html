How to use the Fibonacci web app

{{ msg }}

Hello and welcome to this web app, which calculates a series of Fibonacci numbers.

To use the app, just add the number of Fibonacci numbers you want to calculate to the end of the above url.
i.e.: <domain>/fib/<num>

where "<domain>" is the domain and "<num>" is the number of Fibonacci numbers you want to see.

Alternatively, you can see a sample test script.


Enjoy!
